Game play

Wall Bit Game is a jump platform based game similar to the likes of popular mobile game Doodle Jump. Players earn points by reaching the next platform whilst avoiding the falling rocks and staying within the frame of the game which requires an increased speed as players progress.

Play To Earn (P2E)

Wall Bit Game players will be able to earn Wall Bit Game (WBG) tokens by completing challenges and competing against one another. In order to take advantage of the P2E aspect of the game players must hold WBG. As the game increases in popularity the price of WBG will also increase allowing for massive potential profits for players. The P2E aspect of the game is currently under development, we plan to release the game and P2E aspect within 4 weeks from launch.

Worldwide Leaderboard

Wall Bit Game will feature a worldwide leader board in which the top 100 players worldwide will be rewarded in Wall Bit Game tokens. The leaderboard rewards will released weekly and will be a set proportion of the reward fees allowing for the rewards to increase as wallbit game popularity grows.
